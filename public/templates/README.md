# Templates Images

Ce dossier contient les images des templates d'agents pré-configurés.

## Instructions

Placez vos 4 images d'avatars africains 3D dans ce dossier avec les noms suivants :

1. **template-1.png** - Avatar masculin avec tenue traditionnelle verte (Kofi - Agent de vente)
2. **template-2.png** - Avatar féminin en tailleur professionnel (Amara - Support client)
3. **template-3.png** - Avatar masculin en costume bleu (Malik - Directeur commercial)
4. **template-4.png** - Avatar féminin avec cheveux bouclés (Élisa - Directrice opérationnelle)

## Format des images

- Format : PNG ou JPG
- Dimensions recommandées : 300x300px minimum
- Fond : Avec gradient coloré
- Style : Avatar 3D africain réaliste

## Utilisation

Les images seront automatiquement chargées et affichées sur la page d'accueil dans la section "Choisir un Template".

Si une image est manquante, un placeholder avec les initiales sera affiché à la place.
